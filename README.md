# Job Portal

A modern full-stack Job Portal application built using the MERN stack, enabling users to search and explore job opportunities through an intuitive and responsive interface.

## Features

* Secure user registration and login using JWT authentication
* Browse available job listings
* Search jobs by keywords
* Filter jobs based on location and job type
* Responsive and mobile-friendly UI
* Protected routes for authenticated users
* Password encryption using bcryptjs

## Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* Axios

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Authentication**

* JSON Web Token (JWT)
* bcryptjs

## Installation and Setup

### Clone Repository

```bash
git clone <repository-url>
cd job-portal
```

### Backend Setup

```bash
cd server
npm install
node index.js
```

### Frontend Setup

```bash
npm install
npm run dev
```
